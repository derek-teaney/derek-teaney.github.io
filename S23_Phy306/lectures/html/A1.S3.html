A.3 The Gamma function

(February 5, 2023)

The Γ⁢(x) function is fundamental, and for z>0 it can be defined as11 1 The reason why I write d⁢x/x is somewhat stylized, and is because I like to see clearly the argument z by itself, and because d⁢x/x is unchanged by a rescaling x→x′=a⁢x, i.e. d⁢x′/x′=d⁢x/x.

Γ⁢(z)=∫0∞d⁢xx⁢e-x⁢xz (A.9)

We have the following facts

  1. 1.

    When z=n is an integer.

    Γ⁢(n)=(n-1)! (A.10)

    Γ⁢(z) is a unique generalization of the factorial function to real and even complex numbers

  2. 2.

    We have:

    Γ⁢(12)=π
  3. 3.

    Γ⁢(z) has a factorial like property:

    Γ⁢(z+1)=z⁢Γ⁢(z) (A.11)

    which can be used to show Γ⁢(3/2)=3⁢π/2.

  4. 4.

    The area of a sphere in d dimensions is

    Ad=2⁢πd/2Γ⁢(d/2)⁢rd-1. (A.12)

    which comes up a lot in this course

  5. 5.

    Since the Γ function is like the factorial function, for large with z≫1 the Stirling approximation applies:

    Γ⁢(z)≃(ze)z (A.13)
